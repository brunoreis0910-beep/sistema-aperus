┌──────────────────────────────────────────────────────────────────────────────┐
│ PASSO 1: Configure o arquivo .env                                           │
└──────────────────────────────────────────────────────────────────────────────┘

Abra o arquivo .env e configure:

  DB_ENGINE=django.db.backends.postgresql
  DB_NAME=aperus_producao              ← Nome do seu banco
  DB_USER=aperus_user                  ← Usuário
  DB_PASSWORD=SuaSenha123              ← Senha
  DB_HOST=localhost                    ← Host (localhost ou IP)
  DB_PORT=5432                         ← Porta PostgreSQL
  
  DEBUG=False                          ← IMPORTANTE! False em produção
  ENVIRONMENT=production
  ALLOWED_HOSTS=192.168.1.10,seudominio.com
  
  SECRET_KEY=                          ← Gere uma chave (veja abaixo)


┌──────────────────────────────────────────────────────────────────────────────┐
│ PASSO 2: Gere uma SECRET_KEY                                                │
└──────────────────────────────────────────────────────────────────────────────┘

PowerShell:

  .\.venv\Scripts\Activate.ps1
  python -c "from django.core.management.utils import get_random_secret_key; print(get_random_secret_key())"

Cole o resultado em SECRET_KEY= no arquivo .env


┌──────────────────────────────────────────────────────────────────────────────┐
│ PASSO 3: Instale o sistema                                                  │
└──────────────────────────────────────────────────────────────────────────────┘

  1. Clique 2x em: INSTALAR.bat
  
  2. Escolha: [A] - Instalação Completa
  
  3. Aguarde a instalação (30-45 minutos)
